# zvnc
a variant of TurboVNC enhancement on user-input and display-output

### Features
* UTF-8 clipboard
* Multiple input: keyboard, mice and TBD: multi-touch, gesture, voice, a/vr and hologram
* Interactive video streaming as display-output
